# The Angular CLI


[Angular.io](https://angular.io/)

[Angular CLI "Ng"](https://angular.io/cli)
## Installing

```shell
npm i -g @angular/cli
```

## Checking Version

```shell
ng version
```

## Creating an application

```shell
ng new APP_NAME
```

## Serving for Development

```shell
ng serve -o
```

> note: the `-o` tells it to open the default browser when it is running.
